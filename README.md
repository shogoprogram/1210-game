# とんぴょんのポーションメーカー (Tonpyon’s Potion Maker)

## 概要

材料を混ぜてポーションやオリジナルの背景色を作る、シンプルなクリッカーゲームです。  
このゲームは、単一の`index.html`ファイルのみで動作します。

## 遊び方

### 1. タイトル画面
- **Start:** ゲーム画面を開始します。
- **Settings:** 設定メニューを開きます。

### 2. ゲーム画面
- **材料の選択:**
  - 画面に表示されている「色の材料」「特殊材料（草、毒、上級魔法）」「完成ポーション」の中から、３つまで材料を選びます。
  - ４つ以上選ぼうとすると、ポップアップで警告が表示されます。
- **ポーションの生成:**
  - 材料を２つ以上選ぶと「生成」ボタンが押せるようになります。
  - ボタンを押すと、材料が合成されて新しいポーションが「完成ポーション」の欄に追加され、スコアが１増えます。
  - 完成したポーションは、次の合成の材料として使用できます。
- **選択の解除:**
  - **個別解除:** 「選んだ材料」に表示されている小さい丸いアイコンをクリックすると、その材料だけを選択から外せます。
  - **一括解除:** 「全てクリア」ボタンを押すと、選択中の全ての材料を解除します。
- **ポーションの破棄:**
  - 「ポーションを破棄」ボタンを押すと、作成したポーションの一覧が表示され、不要なポーションを個別に削除できます。
- **ゲームのリセット:**
  - 「ゲームリセット」ボタンを押すと、スコアと全ての完成ポーションが初期化されます。
- **タイトルに戻る:**
  - 「戻る」ボタンを押すと、ゲームの進捗（スコア、選択中の材料、完成ポーション）をリセットするか確認するポップアップが表示されます。「OK」を押すと、進捗がリセットされてタイトル画面に戻ります。

### 3. 設定画面
タイトル画面の「Settings」ボタンから移動できます。

- **言語設定 (Language):**
  - 「日本語」と「English」を切り替えることができます。ゲーム内のすべての表示が選択した言語に変わります。
- **背景色設定 (Background Color Settings):**
  - ここでは、アプリ全体の背景色を自由に作成・設定できます。
  - **色の合成:** 基本の６色や、合成して作った色を３つまで選び、「生成」ボタンで新しい色を作成できます。
  - **背景への適用:** 作成した色や元からある色をクリックして選択し、「この色を背景色にする」ボタンを押すと、アプリ全体の背景色が変わります。
  - **色の破棄:** 「色を破棄」ボタンで、作成した色を削除できます。

## 工夫した点

- **分かりやすいUI/UX:**
  - ユーザーの方からのフィードバックを元に、操作に迷わないようなデザインを追求しました。特に、「選んだ材料」を個別に取り消す機能が分かりやすくなるよう、小さい「×」アイコンを表示するなどの工夫を重ねました。
- **画面遷移と状態管理:**
  - ゲームの進行状況（スコアやアイテム）を保持したまま、タイトルや設定画面をスムーズに行き来できます。ゲームの進捗がリセットされる可能性がある操作の前には、確認のポップアップを表示し、意図しないリセットを防ぐようにしました。
- **背景色作成機能:**
  - 単純なカラーピckerではなく、ゲーム本体と同じ「合成」のメカニズムを使って、ユーザー自身が楽しみながら好みの背景色を作成できる「ミニゲーム」として実装しました。
- **柔軟な言語対応:**
  - すべてのUIテキストを動的に切り替えられるように設計し、簡単かつ瞬時に日本語と英語の表示を切り替えられるようにしました。
