# ローエンのミニドラゴンゲーム (Mini Dragon Game in Loen)

## 概要
「ローエンのミニドラゴンゲーム」は、クラシックなスネークゲームにインスパイアされた、ドラゴンを操作するアクションゲームです。プレイヤーはドラゴンを操作し、フィールド上のアイテムを収集しながら、敵を避けたり倒したりしてスコアを競います。肉を食べてドラゴンを成長させ、マッシュルームで体を縮め、スペースキーで攻撃を放ちましょう。

## 遊び方（操作方法）

### 基本操作
-   **移動**: **WASD** キー
    -   W: 上へ移動
    -   A: 左へ移動
    -   S: 下へ移動
    -   D: 右へ移動
-   **攻撃**: **スペースキー**
    -   ドラゴンの向いている方向に攻撃を放ち、敵を倒すことができます。

### アイテムと効果
フィールド上には様々なアイテムが一定時間で出現します。
-   **肉 (Meat)**: ドラゴンの体が成長し、攻撃範囲が1マス増えます。
-   **マッシュルーム (Mushroom)**: ドラゴンの体が縮み、攻撃範囲が1マス減ります（最小1マス）。
-   **？肉 (Wonder Meat)**: 取得するとゲームオーバーになります。
-   **敵 (Enemy)**: 触れるとゲームオーバーになります。攻撃で倒すことができます。
-   **コイン (Coin)**: 取得しても特別な効果はありませんが、スコアが加算されます。

### ゲームのルール
-   壁や自身の体に触れるとゲームオーバーです。
-   ？肉や敵に触れてもゲームオーバーです（敵は攻撃で倒せます）。
-   一定の時間がたつとフィールド上のアイテムが新しいものに変わります。
-   ドラゴンの移動はキー入力ごとに1マスずつ行われます。キーを押しっぱなしにしても連続で移動することはありません。

## 工夫した点

-   **英語にも対応**: 日本語でなく英語にも対応し設定から言語を選択することが可能です。
-   **色カスタマイズ**: 背景色、ドラゴンの色、ドラゴンの攻撃色をプレイヤーが自由に設定できます。
-   **動的な難易度設定**: 5x5から30x30まで、5段階のボードサイズから難易度を選択できます。
-   **視覚的なフィードバック**:
    -   ドラゴンの頭には、進行方向を示す2つの目玉と「me」の表示があります。
    -   アイテムは、その種類を示すテキストがセル内に動的に調整されたフォントサイズで表示され、視認性が向上しています。
-   **直感的なUI**: 各画面のボタン配置や画面遷移を分かりやすく設計しました。
